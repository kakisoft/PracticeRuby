## 小リファレンス
ri （使用例：ri Array）
　  
　  

|  演算子  | 意味 |   使用例      |    演算結果  |
|:-------:|:---------:|:---------:|:---------:|
|  //     |   商      |  10 // 3  | 3         |
|  %      |   あまり  |  10 % 3   | 1         |
|  **     |   べき乗  |  3 ** 3   | 27         |

　  
　  
　  

|  記号   |   ファイル編集モード  |
|:-------:|:--------------------|
|  r     |   読み込み            |
|  w     |   上書き              |
|  a     |   追記                |
